if there is a `main.go` in any of the sub-directories e.g. concurrency/ or interfaces/:
```go
go run main.go
```

if there is a single `.go` file e.g. my-runes/
```go
go run <name>.go
```

otherwise cd into a sub-directory
```go
go build
```
